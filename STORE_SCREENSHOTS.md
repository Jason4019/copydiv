# 商店截图准备指南

本文档说明如何为 Chrome Web Store / Edge Add-ons 准备扩展截图。

## 截图要求

### 必需截图
- **至少 1 张**，建议 3-5 张
- **尺寸**：1280 x 800 像素或 640 x 400 像素
- **格式**：PNG 或 JPEG
- **文件大小**：每张不超过 1MB

## 建议的截图内容

### 1. 主功能展示（必需）
**文件名**：`screenshot-1-main-feature.png`
- 展示 popup 弹窗界面
- 显示常用词列表（横向排列，带颜色标记）
- 展示分类标签
- 突出核心功能：快速复制常用词

### 2. 选项页面 - 常用词管理
**文件名**：`screenshot-2-words-management.png`
- 展示选项页面的"常用词管理"标签
- 显示添加、编辑、删除功能
- 展示分类管理和颜色标记功能

### 3. 选项页面 - 配置管理
**文件名**：`screenshot-3-settings.png`
- 展示"配置管理"标签
- 显示语言切换功能
- 显示默认分类设置

### 4. 多行文本支持
**文件名**：`screenshot-4-multiline.png`
- 展示支持多行文本的常用词
- 展示文本自动换行效果

### 5. 帮助页面（可选）
**文件名**：`screenshot-5-help.png`
- 展示内置帮助文档
- 展示使用说明

## 截图制作步骤

### 方法 1：浏览器截图工具
1. 安装扩展并加载到浏览器
2. 打开 popup 或 options 页面
3. 使用浏览器开发者工具（F12）的截图功能
4. 或使用系统截图工具（Windows: Win+Shift+S, Mac: Cmd+Shift+4）

### 方法 2：使用扩展页面直接截图
1. 打开 `edge://extensions/` 或 `chrome://extensions/`
2. 找到 CopyDiv 扩展
3. 点击"详细信息"或"选项"
4. 在新打开的页面中截图

### 方法 3：使用在线工具
- 使用 Figma、Canva 等设计工具制作截图
- 添加标注和说明文字

## 截图优化建议

### 视觉优化
- ✅ 使用清晰的字体和合适的字号
- ✅ 确保文字可读
- ✅ 使用合适的颜色对比度
- ✅ 添加简洁的标注（如箭头、文字说明）

### 内容优化
- ✅ 展示真实的使用场景
- ✅ 使用有意义的示例数据
- ✅ 突出核心功能
- ✅ 展示界面美观性

### 技术优化
- ✅ 确保截图清晰（高分辨率）
- ✅ 压缩文件大小（使用工具如 TinyPNG）
- ✅ 保持一致的风格

## 文件命名规范

建议使用以下命名规范：
- `screenshot-1-main-feature.png`
- `screenshot-2-words-management.png`
- `screenshot-3-settings.png`
- `screenshot-4-multiline.png`
- `screenshot-5-help.png`

## 存储位置

建议在项目根目录创建 `screenshots/` 文件夹：
```
copydiv/
  ├── screenshots/
  │   ├── screenshot-1-main-feature.png
  │   ├── screenshot-2-words-management.png
  │   └── ...
  └── ...
```

## 注意事项

1. **隐私**：确保截图中的示例数据不包含真实个人信息
2. **版权**：如果使用第三方图标或素材，确保有使用许可
3. **一致性**：所有截图应保持一致的风格和色调
4. **更新**：当界面更新时，记得更新截图

## 工具推荐

- **截图工具**：Snagit, Lightshot, ShareX
- **图片编辑**：GIMP, Paint.NET, Photoshop
- **图片压缩**：TinyPNG, ImageOptim
- **标注工具**：Annotate, Snagit

---

**提示**：在提交到商店前，建议在不同浏览器和操作系统上测试扩展，确保截图准确反映实际使用体验。
